# Aplicacao fullstack - basica

> Uma aplicação web full-stack, construída do zero com uma stack de tecnologias padroes. Este projeto serve como uma peça de portfólio profissional, demonstrando um ciclo de desenvolvimento completo, desde o design do banco de dados com Prisma até uma interface de usuário reativa. O back-end é uma API RESTful alimentada por Node.js e Express, projetada para escalabilidade e manutenibilidade.
